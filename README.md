# 🎲 Number Guessing Game

![Game Screenshot](screenshot.png)

> A fun number guessing game built with Python! Try to guess the secret number between 1-10. ✨

---

## 🎮 How to Play

1. 🏃‍♀️ Run the game
2. 🔢 Guess a number (1-10)
3. 📢 Get hints: "Too high" ⬆️ or "Too low" ⬇️
4. 🎉 Keep guessing until you win!
5. 🚪 Type `quit` to exit

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎲 | Random number each round |
| 🔢 | Tracks your attempts |
| 💬 | Interactive hints |
| 🔄 | Play multiple rounds |
| 🚪 | Exit anytime |

---
